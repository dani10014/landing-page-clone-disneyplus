# Clone Disney+ | Estudo de Interface Visual 

Estava dando uma geral nos meus arquivos e me deparei com um projeto antigo. Esse clone da Disney+ era para ser um trabalho da EBAC, onde o professor mostrava o passo a passo de como fazer. 

Mas a verdade é que eu não me sinto confortável apenas seguindo receita de bolo de professor. Eu queria me desafiar de verdade. 

O que eu fiz? Abri a página oficial da Disney+, puxei o editor de código e comecei a programar exatamente o que eu estava vendo na minha tela, do zero, sem olhar nenhuma vez as ferramentas de desenvolvedor ou os estilos aplicados no site deles. Refiz tudo no olho!

O resultado foi esse do vídeo. Ficou absurdamente fiel ao design oficial (versão 2026), com os efeitos de hover nos cards do Top 10 e a transição dos blocos de marcas. Ainda tem alguns pequenos detalhes de responsividade para eu lapidar nas telas menores, mas o aprendizado de desbravar o CSS na raça não tem preço.
